# Bigxyt-Task
## General info
Simple orderbook like python application
## Technologies
Python
## Setup
To run this app simple write in console:
```
$ python big_xyt.py
```
There will be 4 options to choose
```
1 - Create order
2 - Display orders
3 - Find best prices
q - Quit
```
